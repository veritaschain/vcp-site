VCP Explorer API v1.1

暗号学的証明によるVCPイベントチェーンの検索、検証、監査

GitHubでAPIを見る

概要

VCP Explorer APIは、VeritasChain Protocol (VCP)のイベント検索、Merkle証明取得、イベント証明書生成を提供します。VCP仕様v1.0に完全準拠しており、規制当局、ブローカー、プロップファーム、監査人などの外部関係者が、記録されたイベントを独立して検証できるようにします。

暗号学的検証

RFC 6962準拠のMerkle証明により、APIを信頼せずに外部検証が可能です。「検証せよ、信頼するな」の原則を実践しています。

高度な検索

TraceIDで完全なトランザクションライフサイクルを追跡し、シンボル、時間範囲、イベントタイプ、アルゴリズムIDでフィルタリングできます。

規制対応証明書

イベントデータ、Merkle証明、暗号署名を含む監査対応証明書を生成し、コンプライアンスを実現します。

VCPモジュール構造

VCP-TRADE、VCP-RISK、VCP-GOVモジュールで整理されたイベントデータにより、明確な関心事の分離を実現します。

ナノ秒精度

文字列ベースのタイムスタンプはJavaScriptの2^53制限を回避し、真のナノ秒レベルの精度をサポートします。

ブロックチェーンアンカリング（計画中）

VCP Explorerの将来バージョンでは、Ethereumへの監査ルートのアンカリングをサポートし、追加の不変性保証を提供します。

ベースURL

https://explorer.veritaschain.org/api/v1
注意: Explorer APIのデプロイメントは進行中です。ライブエンドポイントが利用可能になるまで、このドキュメントは実装リファレンスとして機能します。デプロイメントが発表されるまで、開発者はこれらのエンドポイントを呼び出さないでください。

認証

すべてのAPIリクエストは、リクエストヘッダーにAPIキーを使用した認証が必要です:

HTTPヘッダー 
X-VCP-API-Key: <your_api_key>
セキュリティ: APIキーをクライアントサイドコードに公開しないでください。常にサーバーサイドリクエストで適切なキー管理を使用してください。
