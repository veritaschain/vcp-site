VCP Explorer API v1.1

使用加密证明搜索、验证和审计VCP事件链

在GitHub上查看API

概述

VCP Explorer API为VeritasChain协议(VCP)提供事件搜索、Merkle证明检索和事件证书生成功能。它完全符合VCP规范v1.0,使外部方(监管机构、经纪商、自营交易公司、审计师)能够独立验证记录的事件。

加密验证

符合RFC 6962的Merkle证明实现无需信任API的外部验证。践行"验证,不要信任"的原则。

高级搜索

使用TraceID跟踪完整的交易生命周期,按符号、时间范围、事件类型和算法ID进行过滤。

监管证书

生成包含事件数据、Merkle证明和加密签名的审计就绪证书,实现合规性。

VCP模块结构

通过VCP-TRADE、VCP-RISK和VCP-GOV模块组织的事件数据,实现清晰的关注点分离。

纳秒精度

基于字符串的时间戳避免了JavaScript 2^53限制,支持真正的纳秒级精度。

区块链锚定(计划中)

VCP Explorer的未来版本将支持将审计根锚定到以太坊,提供额外的不可变性保证。

基础URL

https://explorer.veritaschain.org/api/v1
注意: Explorer API部署正在进行中。在实时端点可用之前,本文档作为实施参考。开发者不应在部署公告之前尝试调用这些端点。

认证

所有API请求都需要在请求头中使用API密钥进行认证:

HTTP头 
X-VCP-API-Key: <your_api_key>
安全: 不要在客户端代码中暴露API密钥。始终使用服务器端请求和适当的密钥管理。
