利用規約

VCP Explorer / API / SDK

最終更新日：2025年12月1日

1 概要

本利用規約（以下「本規約」）は、VeritasChain Standards Organization（VSO）が提供するVCP Explorer、API、SDK（以下「本サービス」）の利用に適用されます。VSOは、VeritasChain株式会社が運営しています。

本サービスにアクセスまたは利用することにより、お客様は本規約に同意したものとみなされます。本規約に同意いただけない場合は、本サービスのご利用をお控えください。

2 免責事項

重要なお知らせ：本サービスをご利用になる前に、本セクションを必ずお読みください。

2.1 情報提供目的

VCP Explorerを通じて提供されるデータおよび情報は、参考情報として提供されるものであり、以下のものを構成するものではありません：

  • 財務、投資、または取引に関するアドバイス
  • 資産の売買または保有の推奨
  • 専門的な法務、税務、または規制に関するアドバイス
  • 正確性、完全性、または適時性の保証

2.2 「現状有姿」での提供

APIおよびSDKは、「現状有姿（AS IS）」および「利用可能な状態（AS AVAILABLE）」で提供され、明示または黙示を問わずいかなる種類の保証も行いません。

2.3 損害に対する免責

VSOは、以下から生じるいかなる損害または損失についても責任を負いません：

  • 市場損失 — 本サービスのデータに基づく取引損失または投資判断
  • システム障害 — サービスの中断、ダウンタイム、または技術的な問題
  • データ遅延 — 遅延した、不完全な、または不正確なデータ伝送
  • セキュリティインシデント — VSOの合理的な管理を超えた不正アクセスまたはデータ侵害

3 利用者の禁止事項

本サービスの利用者は、以下の禁止行為を行わないことに同意するものとします：

3.1 不正アクセス

ハッキング、パスワード解析、その他の手段により、本サービス、他のアカウント、コンピュータシステム、または本サービスに接続されたネットワークへの不正アクセスを試みること。

3.2 解析・リバースエンジニアリング

本サービスのソースコード、基礎となるアイデア、アルゴリズム、または構造を導き出すための、リバースエンジニアリング、逆コンパイル、逆アセンブル、またはその他の試み。

3.3 システムへの過負荷

インフラストラクチャに不合理または過度な負荷をかける行為：

  • レート制限を超える過度なAPIコール
  • サービス妨害攻撃
  • 許可なき自動スクレイピング
  • 本サービスを妨害または干渉するあらゆる活動

3.4 なりすまし

他の個人または団体になりすますこと、または個人または団体との関係について虚偽の陳述または不実表示を行うこと。

4 知的財産権

本サービスに関するすべての知的財産権は、VSOまたはそのライセンサーに帰属します。

4.1 保護対象

  • VCP仕様書 — 技術仕様書および文書
  • SDKおよびライブラリ — ソフトウェア開発キットおよびコードライブラリ
  • VCP Explorer UI — ユーザーインターフェースのデザインおよび実装
  • 商標 — VeritasChain、VCP、VC-Certified、および関連ロゴ

4.2 制限事項

  • 事前の書面による同意なく、本サービスの一部を再配布、サブライセンス、または再販売すること
  • 本サービスに関する所有権表示またはラベルを削除または改変すること
  • 明示的な許可なくVSOの商標を使用すること

4.3 オープンソースコンポーネント

SDKの特定のコンポーネントは、オープンソースライセンス（Apache 2.0またはMITライセンス）の下でリリースされる場合があります。

5 責任の制限

5.1 責任の上限

VSOの累積責任総額は、以下のいずれか大きい方を超えないものとします：

  • 請求に先立つ12ヶ月間にお客様がVSOに本サービスのために支払った総額、または
  • 本サービスが無料で提供された場合はゼロ（0）円

6 準拠法および管轄

6.1 準拠法

本規約は、日本法に準拠し、日本法に従って解釈されるものとします。

6.2 管轄裁判所

本規約に起因する紛争については、東京地方裁判所を第一審の専属管轄裁判所とします。

6.3 言語

本規約の英語版と翻訳版との間に相違がある場合は、英語版が優先するものとします。

7 規約の変更

7.1 変更の権利

VSOは、独自の裁量により、いつでも本規約を修正、更新、または改訂する権利を留保します。

7.2 通知

  • VCP Explorerウェブサイト上での告知
  • 本規約冒頭の「最終更新日」の更新
  • 重大な変更の場合のメール通知（メールアドレスを登録している場合）

7.3 変更の承諾

本規約の変更後も本サービスを継続して使用することにより、改訂された規約に同意したものとみなされます。
