1. 収集するデータ

VSO は、サービスご利用時に以下のデータを収集する場合があります：

IPアドレス

セキュリティ監視、不正防止、サービス最適化のためにIPアドレスを収集します。

Cookieデータ

セッション状態の維持、設定の保存、アクセス解析のためにCookieを使用します。

Explorer/SDK/API の利用ログ

運用監視のため、VCP Explorer、SDK、API サービスとのやり取りを記録します。

デバイス情報

ブラウザの種類、オペレーティングシステム、画面解像度などの技術情報。

問い合わせフォームの入力情報

お問い合わせフォームで提供いただく情報（名前、メールアドレス等）。

アクセス解析情報（Google Analytics 4）

サービス改善のため、Google Analytics 4 を通じて匿名化された利用統計を収集します。

2. 利用目的

収集したデータは、以下の目的で利用します：

  • VCP Explorer / API / SDK の安定運用

    サービスの信頼性と効率的な運用の確保、システム状態の監視、パフォーマンスの最適化。

  • 不正アクセス検知、セキュリティ強化

    不正行為、不正アクセスの試み、その他のセキュリティ脅威の検出と防止。

  • プロトコル改善のための統計分析

    利用パターンを分析し、VeritasChain Protocol および関連サービスを改善するため。

  • VC-Certified（認証制度）運用のための最低限の情報管理

    認証プロセスを必要最小限の情報で管理するため。

  • 法令遵守（MiFID II, EU AI Act など）に必要な監査ログ保全

    MiFID II、EU AI Act、その他適用法令に基づく監査ログの維持。

3. Cookie とアクセス解析

Google Analytics 4 (GA4)

当ウェブサイトでは、匿名化された利用統計を収集するために Google Analytics 4（GA4）を使用しています。GA4 は Cookie を使用してユーザーの操作を追跡し、ウェブサイトのトラフィックや利用パターンに関するレポートを生成します。

GA4 で収集される情報

  • • 訪問したページと各ページでの滞在時間
  • • 参照元（どこからアクセスしたか）
  • • ブラウザとデバイスの種類
  • • 地理的位置（国・地域レベル）
  • • ユーザーの操作（クリック、スクロール）

Cookie の設定管理

ブラウザの設定で Cookie を管理・拒否することができます。多くのブラウザでは以下のことが可能です：

  • デバイスに保存されている Cookie の表示
  • 既存の Cookie の削除
  • すべてまたは特定の Cookie のブロック
  • 特定のウェブサイトに対する設定

オプトアウト方法

Google Analytics オプトアウト アドオン をインストールすることで、Google Analytics のトラッキングをオプトアウトできます。

サードパーティ解析サービス

GA4 に加えて、サービス改善のために他のサードパーティ解析サービスを使用する場合があります。これらのサービスは独自の Cookie を設定し、同様のデータを収集する可能性があります。使用するすべてのサードパーティサービスが適用されるプライバシー規制に準拠していることを確認しています。

4. 第三者への提供

個人情報の販売はしません

VSO は、マーケティングやその他の商業目的で、お客様の個人情報を第三者に販売、賃貸、または取引しません。

データを共有する場合

お客様のデータは、以下の限定的な状況でのみ共有される場合があります：

  • 法的要請

    法律、裁判所の命令、または規制当局から要求された場合（例：法執行機関からの要請、規制調査）。

  • VC-Certified 監査目的

    VC-Certified プログラムの監査目的で、必要最小限の匿名統計データが使用される可能性があります。このデータには個人を特定できる情報は含まれません。

データの保存期間

本ポリシーに記載された目的を達成するために必要な最小限の期間（最大24ヶ月）のみ、個人データを保持します。法律（規制遵守要件など）でより長い保持期間が要求される場合を除きます。

GDPR に基づくユーザーの権利

EU 居住者として、個人データに関する以下の権利を有しています：

アクセス権

個人データへのアクセスを要求し、保有するデータのコピーを受け取る権利。

訂正権

不正確または不完全な個人データの訂正を要求する権利。

削除権

特定の条件下で個人データの削除を要求する権利。

処理制限権

特定の条件下で処理の制限を要求する権利。

処理の法的根拠

GDPR 第6条に基づき、サービスの運営、改善、セキュリティ確保における正当な利益を根拠として個人データを処理します。ただし、そのような処理がお客様の基本的権利と自由を侵害しないことを確保します。特定の処理活動については、お客様の同意、契約上の必要性、または法的義務に基づく場合があります。

6. セキュリティ

不正アクセス、改ざん、開示、破壊から個人データを保護するため、堅牢なセキュリティ対策を実施しています。

HTTPS通信

ブラウザとサーバー間のすべての通信は HTTPS（TLS/SSL）で暗号化されています。

業界標準暗号化

VCP Explorer と API は、転送中および保存時のデータを保護するため業界標準の暗号化プロトコルを使用しています。

内部アクセス制御

内部システムにはロールベースのアクセス制御（RBAC）と最小権限の原則を適用しています。

7. ポリシーの変更

当社の慣行、技術、法的要件、その他の要因の変更を反映するため、本プライバシーポリシーを随時更新する場合があります。

変更の通知

個人データの取り扱いに重大な影響を与える変更については、変更が有効になる前に、ウェブサイトまたは適切な手段で通知します。
