1. 我们收集的数据

VSO 在您使用我们的服务时可能会收集以下类型的数据：

IP 地址

为了安全监控、欺诈防范和服务优化，我们会收集您的 IP 地址。

Cookie 数据

Cookie 用于维护会话状态、保存偏好设置和提供分析功能。

Explorer/SDK/API 使用日志

为运营监控目的，记录您与 VCP Explorer、SDK 和 API 服务的交互日志。

设备信息

浏览器类型、操作系统、屏幕分辨率和其他技术数据。

联系表单提交信息

通过联系表单提供的信息，包括姓名和电子邮件地址。

分析数据（Google Analytics 4）

通过 Google Analytics 4 收集的匿名使用统计数据，用于服务改进。

2. 使用目的

我们收集的数据用于以下目的：

  • VCP Explorer / API / SDK 的稳定运营

    确保服务可靠高效运行，监控系统健康状况，优化性能。

  • 未授权访问检测与安全加强

    检测和防止欺诈活动、未授权访问尝试和其他安全威胁。

  • 协议改进的统计分析

    分析使用模式，改进 VeritasChain Protocol 及相关服务。

  • VC-Certified 认证计划管理

    以必要的最少信息管理认证流程。

  • 法规遵从与审计日志保留

    维护符合 MiFID II、EU AI Act 等适用法规要求的审计日志。

3. Cookie 与分析

Google Analytics 4 (GA4)

我们的网站使用 Google Analytics 4（GA4）收集匿名使用统计数据。GA4 使用 Cookie 跟踪用户交互并生成网站流量和使用模式的报告。

GA4 收集的信息

  • • 访问的页面和在每个页面停留的时间
  • • 引荐来源（您从哪里来）
  • • 浏览器和设备类型
  • • 地理位置（国家/地区级别）
  • • 用户交互（点击、滚动）

管理 Cookie 偏好

您可以通过浏览器设置控制和拒绝 Cookie。大多数浏览器允许您：

  • 查看设备上存储的 Cookie
  • 删除现有 Cookie
  • 阻止所有或特定 Cookie 的设置
  • 为特定网站设置偏好

退出选项

您可以通过安装 Google Analytics 选择退出浏览器插件 来退出 Google Analytics 跟踪。

第三方分析服务

除 GA4 外，我们可能使用其他第三方分析服务来改进我们的服务。这些服务可能设置自己的 Cookie 并收集类似数据。我们确保使用的所有第三方服务符合适用的隐私法规。

4. 第三方共享

不出售个人信息

VSO 不会出于营销或任何其他商业目的向第三方出售、出租或交易您的个人信息。

我们可能共享数据的情况

您的数据仅在以下有限情况下可能被共享：

  • 法律要求

    当法律、法院命令或监管机构要求时（例如：执法请求、监管调查）。

  • VC-Certified 审计目的

    VC-Certified 计划的审计目的可能使用最少的匿名统计数据。此类数据不包含个人身份信息。

6. 安全措施

我们实施强有力的安全措施，保护您的个人数据免受未经授权的访问、更改、披露或破坏。

HTTPS 加密

您的浏览器与我们服务器之间的所有通信均使用 HTTPS（TLS/SSL）加密。

行业标准加密

VCP Explorer 和 API 使用行业标准加密协议保护传输中和静态数据。

访问控制

对内部系统应用基于角色的访问控制（RBAC）和最小权限原则。
