Frequently Asked Questions

How do I start implementing VCP?

We recommend starting with Silver Tier Shadow Logging. This requires minimal changes to your existing system and provides immediate tamper-proof audit trails.

View Quick Start Guide

What is the VC-Certified application process?

Submit your implementation details, run our conformance test suite, and undergo technical review. Typical timeline is 2-4 weeks for Silver/Gold, 6-8 weeks for Platinum.

View Certification FAQ

Does VCP work with my existing FIX infrastructure?

Yes! VCP uses a "sidecar" pattern that runs alongside your FIX engine without modifications. We provide adapters for all major FIX implementations.

View Integration Patterns

What is the cost structure?

VCP specification and reference SDKs are free and open-source (CC BY 4.0). VC-Certified audits and VCP Cloud services have usage-based pricing. Contact us for enterprise quotes.
